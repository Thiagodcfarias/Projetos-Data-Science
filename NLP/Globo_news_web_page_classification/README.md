# Introdução

## Objetivo

1. Coletar dados de textos dos sites g1,ge e gshow 
2. Classificá-los quanto a sua origem

## Fontes:

1. https://g1.globo.com/
2. https://gshow.globo.com/
3. https://globoesporte.globo.com/

## Resultados:

82% de acurácia com desvio padrão de 0.03
